During my lessons, I have learned the importance of Web Standards and HTML. Web standards are used to ensure better accessibility, performance, and security for end-users, this includes the use of HTML (HyperText Markup Language). HTML is used in creating websites by providing a standard structure, used worldwide. This is helpful so that developers who want to create a website will follow the same standards, giving consistency.

What I Learned about Web Standards:

  • Consistency
  • Accessibility
  • Performance
  • Security

What I Learned about HTML:

  • Provides a fundamental structure.
  • Uses element tags to show how content is shown

Lab 3

Published on February 21,2025 by Student, A.Chung

In this module I learned about CSS (Cascading Style Sheets). HTML is used to structure your webpage, while CSS is used to manipulate it's style. I knew about external and inline but there is also embedded or internal, which is place inside the head. I find that both embedded and internal is good for simple webpages with maybe one or two pages, but for websites that have mulptiple, external method can neatly organize all your styles in a file. To target certain elements you can use the class and id attribute.

What I Learned about CSS:

  • CSS syntax: Selector: { property ; value }
  • Styles go in cascading order by highest priority
    1st: Inline style
    2nd: Embedded styles
    3rd: External Style Sheet
    Even if you have a external style sheet the Inline style is top priority and will be the style shown on website. (Example: Lab 3 header, internal is red, and external h2 is set to white like Lab 1 & 2.)
