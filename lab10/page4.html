Web Share API

Web standards

The Web Share API is only one of several standards. By conforming to standards, developers can create websites that use technologies that are supported by several browsers. This lowers the likelihood of websites becoming obsolete or broken as browsers update. Many web standards contain instructions for making websites more accessible to individuals with impairments. This could include providing alternative language for images so that screen readers can describe them, as well as ensuring that websites can be accessed solely by keyboard. Web standards frequently represent the best practices for creating effective, secure, and user-friendly websites. Web Share API - W3C Recommendation.
