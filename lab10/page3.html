Web Share API

How is it used?

The Web Share API is generally used with the navigator.share() method, and to determine whether the API is supported, use navigator.canShare(). There is a feature called "web-share" that inhibits unwanted sharing initiation. The share feature is usually associated with a button element. Web Share API - W3C Recommendation.

Here is an example how it is written with JS

            
              if (navigator.share)
                document.getElementById('share').addEventListener('click', () 
                => navigator.share({ title: 'Check this!', url: location.href }));
              else
                document.getElementById('share').style.display = 'none';
