Reflection

In this module, I learned about CSS (Cascading Style Sheets). HTML is used to structure your webpage, while CSS is used to style. I knew about external and inline, I learned a new method, which is embedded/internal, CSS code that is placed inside the head section. Embedded and internal are good for simple webpages with maybe one or two pages, but for websites with multiple, external methods can neatly organize all your styles in a file. To target certain elements you can use the class and id attribute.

What I Learned about CSS:

  • CSS syntax: Selector: { property ; value }
  • Styles go in cascading order by highest priority
    1st: Inline style
    2nd: Embedded styles
    3rd: External Style Sheet
    Even if you have a external style sheet the Inline style is top priority and will be the style shown on website. (Example: Lab 3 header, internal is red, and external h2 is set to white like Lab 1 & 2.)
