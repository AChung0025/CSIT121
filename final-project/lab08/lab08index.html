In this lesson, we are introduced to JavaScript and CSS Anitmation. JavaScript is much like Python and other programming languages, where you are able to create functions for specific elements or actions, that can be reused. This enables user to be able to interact with site. CSS Animation is used with transition or animating HTML elements. Transition lets you transform the "CSS property you want to add an effect to the duration of the effect" (w3schools), with animation you can set multiple points using keyframes.
