GIF

GIF stands for Graphics Interchange Format
The file extension for a GIF is .gif, GIFs can be either still or animated graphics. This image was shown to me in the All About Standard Web Graphics lesson. It is of a one eyed orange monster bouncing up and down, while in the background, there is a color wheel spinning clockwise. I decided to use this image because it is a great example of what a GIF image looks like. This displays the perfect-loop GIF , where the animation at the start is perfectly synced with the ending to make it appear as if it is constant. GIFs are great for expressing reactions and are used for marketing and visual cinemographs.
