Web Share API

What is it?

Web Share API is a web platform API (Application programming interface) that allows users of websites and web applications to share content with other users. The API provides a standardized way to prompt the system's built-in share dialog. This dialog then presents the user with a list of available "share targets," including native applications, web services, contacts, and more. The API enables the sharing of text, URLs, and even files. Web Share API is supported by all major browser engines. Web Share API - W3C Recommendation
